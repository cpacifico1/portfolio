Character Animation

Project Summary

The purpose of this project was to showcase my use of the basics involved in using Adobe Animate and how those skills can be utilized to create a basic animation. I used many tools from Adobe Animate to complete this project. The main tools I used were the Oval Primitive tool and the Classic Brush tool.
